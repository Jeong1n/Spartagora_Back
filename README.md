## 프로젝트 주제 (스파르타고라)
* sparta + agora = spartagora
   * 스파르타코딩클럽 과정을 수강중 혹은 수료한 학생들의 익명 커뮤니티
   * 익명성이 보장된 상태에서 다양한 주제를 관해 이야기하는 커뮤니티
